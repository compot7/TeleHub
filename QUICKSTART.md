# Быстрый старт

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Применение миграций

```bash
python manage.py makemigrations
python manage.py migrate
```

## 3. Создание суперпользователя

```bash
python manage.py createsuperuser
```

Введите имя пользователя, email и пароль для администратора.

## 4. Запуск сервера

```bash
python manage.py runserver
```

Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/

## 5. Первые шаги

1. Зарегистрируйте нового пользователя через `/register/`
2. Войдите в личный кабинет через `/dashboard/`
3. Заполните профиль в разделе "Редактировать профиль"
4. Добавьте проекты в портфолио
5. Добавьте отзывы
6. Добавьте социальные ссылки
7. Просмотрите свою визитку по адресу `/card/<ваш_username>/`

## Административная панель

Доступна по адресу: http://127.0.0.1:8000/admin/

Используйте учетные данные суперпользователя для входа.

## Примечания

- Для работы карты Google Maps необходимо добавить API ключ в шаблон `templates/cards/card_detail.html` (строка ~400)
- Все загруженные файлы сохраняются в папке `media/`
- Статические файлы собираются в `staticfiles/` при использовании `collectstatic`
